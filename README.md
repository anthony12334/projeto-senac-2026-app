# viajei
o objetivo do projeto **Viajei** é, compartilhar das experiências que vivemos, para que outras pessoas também possam usufruir da mesma experiência. essas experiências podem ser sobre qualquer tema (livros, séries, filmes cozinha, esportes, viagens..).

> [!WARNING]
> 
 >o **Viajei**, por razões apropriadas,
 > é indicando para maiores de 14 (quartorze) anos.
 > **não recomendamos** que pessoas com menor idade
 >  do indicado, tenham acesso à plataforma.

## o que será?

o viajei será uma plataforma em que os usuários poderão, indiviualmente, compartilhar suas experiências, também, recomendar outras tantas para as pessoas que acessarem essa plataforma.

>[!important]
> 
>  A plataforma **não** suporta múltiplos usuários.
>  isto não é uma rede social virtual.
>  para compartilhar
>  suas peripécias, por favor,
>  compartilhe o link da sua história.
